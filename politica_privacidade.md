# Política de Privacidade (Gaia OS)

Última atualização: 2025-12-10

Este documento explica, de forma simples, quais dados o aplicativo coleta e como são usados.

1. Dados coletados
- Não coletamos dados pessoais obrigatórios pelo funcionamento do app, a menos que o usuário os forneça voluntariamente (por exemplo: nome ou e-mail para suporte).
- Podemos coletar dados técnicos (ex.: tipo de navegador, idioma) para melhorar o funcionamento.

2. Uso dos dados
- Dados fornecidos voluntariamente são usados apenas para fornecer suporte, responder mensagens ou melhorar o serviço.
- Dados técnicos são usados apenas para analisar erros e melhorar a experiência.

3. Compartilhamento
- Não vendemos seus dados.
- Não compartilhamos dados pessoais com terceiros, exceto quando exigido por lei ou com serviços que você autorize.

4. Armazenamento e segurança
- Mantemos os dados pelo tempo necessário para fornecer o serviço.
- Tomamos medidas básicas de segurança para proteger as informações.

5. Contato
- Para dúvidas ou pedidos relacionados à privacidade, entre em contato via: suporte@gaia.ai
