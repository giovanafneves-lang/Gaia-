# Gaia Financial OS

Plataforma de Alta Performance para Inteligência Financeira e Growth Hacking.

## Funcionalidades
- Dashboard Financeiro em Tempo Real
- Agente IA (Gemini 2.5 Flash)
- Gaia Growth Engine (Marketing Automatizado)
- Laboratório de Backtest
- Social Neural Copy-Trading

## Como Instalar (PWA)
Este aplicativo é um PWA (Progressive Web App). Você pode instalá-lo diretamente pelo navegador:

1. Acesse a URL do aplicativo.
2. No Android (Chrome): Toque no menu (3 pontos) > "Adicionar à tela inicial" ou "Instalar aplicativo".
3. No iOS (Safari): Toque no botão Compartilhar > "Adicionar à Tela de Início".

## Desenvolvimento
Stack: React + Vite + TailwindCSS + Google GenAI SDK.

Build: `npm run build`
